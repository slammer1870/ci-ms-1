Features

Articles

Courses

Curriculums

Articles can be used to engage your audience on a regular basis.

You can write articles in the markdown editor, you can also include image and video embeds

Our courses feature allows you to sell courses directly from your app.

Courses are structured in chapters and lessons.

Curriculums allow you to provide personalised coaching, direct to consumer.

Curriculums are structured the same way as courses but can be assigned to specific users.

Technologies

We use Next JS on the frontend for fast, efficient, server side rendered content that is PWA compatible

We use Magic for a passwordless user authentication experience.

We Use Strapi for a dynamic CMS built with Node JS.

We use Stripe for secure payments with the Stripe Checkout.
